Prueba de Angular Pipe

Definición: Los pipe's son usados para transformar datos facílmente.

DatePipe

  • date

    La fecha de hoy es {{ today | date }}

  • date: 'short'

    La fecha de hoy es {{ today | date: 'short' }}

  • date: 'shortDate'

    La fecha de hoy es {{ today | date: 'shortDate' }}

  • date: 'shortTime'

    La fecha de hoy es {{ today | date: 'shortTime' }}

  • {{ '{ today | date: cccc, LLLL d } of { today | date: yyyy } at { today | date: shortTime }' }}

    La fecha de hoy es {{ today | date: 'cccc, LLLL d' }} of {{ today | date: 'yyyy' }} at {{ today | date: 'shortTime' }}

CurrencyPipe

  • currency

    Mi ahorro es de {{ money | currency }}

  • currency: 'CAD'

    Mi ahorro es de {{ money | currency: 'CAD' }}

  • currency: 'USD'

    Mi ahorro es de {{ money | currency: 'USD' }}

  • currency: 'COP'

    Mi ahorro es de {{ money | currency: 'COP' }}

NumberPipe

  • number

    El número es {{ mynumber | number }}

  • number: '1.2-5'

    El número es {{ mynumber | number: '1.2-5' }}

  • number: '1.4-7'

    El número es {{ mynumber | number: '1.4-7' }}

  • number: '4.2-2'

    El número es {{ mynumber | number: '4.2-2' }}

  • number: '1.1-5':'fr'

    El número es {{ mynumber | number: '5.1-5':'fr' }}

PluralPipe

  • i18nPlural: messageMapping

    Mensajes: {{ messages.length | i18nPlural: messageMapping }}

  • i18nPlural: trafficLightMapping

    Se debe {{ trafficLight | i18nPlural: trafficLightMapping }}

SelectPipe

  • i18nSelect: inviteMap

    Segun genero se dice {{ gender | i18nSelect: inviteMap }}

JsonPipe

  • Within pipe

    El objeto es {{ object }}

  • With pipe

    El objeto es {{ object | json }}

LowerCasePipe

  • lowercase

    Mi nombre es {{ myName | lowercase }}

  • lowercase

    Mi nombre es {{ myName | uppercase }}

PercentilePipe

  • percent

    La bateria está en {{ batteryLevel | percent }}

TitlePipe

  • titlecase

    El titulo del articulo es {{ myTitle | titlecase }}
